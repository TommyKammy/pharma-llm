# Experiment Reports

Baseline reports are generated by:

```bash
uv run python scripts/generate_baseline_report.py
```

Default output:

```text
results/reports/baseline_report.md
```

Reports generated from CI-safe mock fixtures validate workflow shape only. Use
live baseline outputs before interpreting model quality or Phase 6 LoRA deltas.

Base-vs-LoRA comparison reports are generated by:

```bash
uv run python scripts/generate_lora_comparison_report.py
```

Default output:

```text
results/reports/lora_comparison_report.md
```

The LoRA comparison report rejects mismatched eval ID sets and keeps
safety/style regression notes visible even when scoring is manual-review-only.
